Computer architecture is the foundation upon which all computing systems are built. This comprehensive lecture series has been developed over years of teaching experience at the University of Peradeniya, refined through continuous interaction with students and industry feedback. The materials presented here represent a careful balance between theoretical foundations and practical applications.

The journey through computer architecture begins with fundamental abstractions and progressively builds towards understanding complex systems including multicore processors, cache hierarchies, and virtual memory. Each lecture is designed to be both self-contained and part of a larger narrative that reveals how modern computers achieve their remarkable capabilities.

These materials are intended for undergraduate students in computer engineering and computer science, as well as professionals seeking to deepen their understanding of computer systems. The accompanying video lectures provide additional context and explanations that complement the written notes.

Learning Methods

This course employs a multi-modal learning approach designed to maximize understanding and retention through video lectures, written notes, and practical labs...

How to Use These Materials

This course employs a multi-modal learning approach designed to maximize understanding and retention:

  • Video Lectures: Watch the corresponding YouTube video for each lecture to get a comprehensive explanation with visual aids and real-time problem-solving demonstrations.
  • Written Notes: Use these detailed notes as reference material during and after watching videos. They include additional examples and clarifications.
  • Practical Labs: Apply concepts through hands-on programming exercises in ARM assembly and Verilog HDL.
  • Active Learning: Pause videos to work through examples independently before seeing solutions.
  • Spaced Repetition: Review previous lectures periodically to reinforce foundational concepts.

Recommended Study Approach

  1. Read the lecture notes introduction to understand key topics
  2. Watch the video lecture, taking brief notes on unclear concepts
  3. Re-read the detailed notes, focusing on sections that need clarification
  4. Complete any associated laboratory exercises
  5. Review key takeaways and summary sections
  6. Attempt practice problems and discuss with peers

Introduction

Computer architecture is the science and art of designing computer systems. It sits at the critical intersection of hardware and software, defining how processors execute instructions...

Computer architecture is the science and art of designing computer systems. It sits at the critical intersection of hardware and software, defining how processors execute instructions, how memory is organized, and how data flows through a system. Understanding computer architecture is essential for anyone who wants to write efficient code, design new computing systems, or optimize existing ones.

What You Will Learn

This lecture series covers the complete spectrum of computer architecture topics:

  • Foundations: Computer abstractions, technology trends, and performance analysis
  • Instruction Set Architecture: The ARM architecture and assembly programming
  • Processor Design: Datapath, control units, and pipelining
  • Memory Systems: Cache hierarchies, virtual memory, and optimization techniques
  • Advanced Topics: Multiprocessors, parallel computing, and I/O systems

Prerequisites

Students should have a basic understanding of digital logic design and programming in at least one high-level language (C, Python, Java, etc.). Familiarity with binary number systems and Boolean algebra is helpful but will be reviewed as needed.

Learning Outcomes

Upon completing this course, you will be able to:

  • Understand how high-level programs are translated into machine instructions
  • Write and debug programs in ARM assembly language
  • Design and implement a functional processor using hardware description languages
  • Analyze and optimize computer system performance
  • Understand modern processor features including pipelining, caching, and virtual memory
  • Make informed decisions about hardware-software trade-offs

About Practicals on Computer Architecture

Hands-on practical series to complement the lecture series, providing practical experience processor microarchitecture and memory systems using Verilog. Students build their own processors and implement memory hierarchy with caching.

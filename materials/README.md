# CO224 Materials Folder

This folder contains downloadable materials for the CO224 Computer Architecture course.

## Contents

- `CO224-Complete-Notes.pdf` - Complete lecture notes in PDF format
- `supplementary.zip` - Supplementary materials including code examples, diagrams, and additional resources

## Note

Please place your PDF and supplementary materials in this folder before deploying to GitHub Pages.
The download links on the website will point to these files.
